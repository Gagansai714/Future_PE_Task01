# Call-to-Action (CTA) Prompt

## Objective

Generate persuasive call-to-action sections that encourage visitors to take action.

---

## Prompt

```text
Act as a conversion copywriter.

Create multiple website CTAs for Brew Buzz Cafe.

Generate CTAs for:

- Homepage
- Menu Page
- Reservation Section
- Contact Page
- Footer

Each CTA should include:

- Headline
- Supporting Text
- Button Text

Tone:
Friendly
Welcoming
Action-Oriented
Modern

Avoid aggressive sales language.

Focus on encouraging customer engagement.
```

---

## Expected Output

* Homepage CTA
* Reservation CTA
* Contact CTA
* Footer CTA
